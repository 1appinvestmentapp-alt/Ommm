# APSO Investment App

## How to create a Git Repository

1.  **Initialize Git**
    Open your terminal in this folder and run:
    ```bash
    git init
    ```

2.  **Add Files**
    Stage all the files for the first commit:
    ```bash
    git add .
    ```

3.  **Commit Code**
    Commit the files to your local history:
    ```bash
    git commit -m "Initial commit of APSO App"
    ```

4.  **Connect to GitHub (Optional)**
    If you have created a repository on GitHub, link it:
    ```bash
    git remote add origin https://github.com/YOUR_USERNAME/apso-app.git
    git branch -M main
    git push -u origin main
    ```

## Features
- User Authentication (Login/Register with Referral)
- Investment Plans (Tiered System)
- Wallet Management (Deposit/Withdraw)
- Team & Referral System (3 Levels)
- Admin Dashboard

## Tech Stack
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons
- React Router DOM
